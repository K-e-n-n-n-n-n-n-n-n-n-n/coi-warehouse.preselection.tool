Warehouse Automation Selector Final

Final pre-feasibility screening tool for Singapore warehouses. Uses Excel-controlled scoring weights, feasibility gates, data-readiness checks, technology-specific constraints, realistic cost ranges, ranking explanations, benchmark validation and confidence scoring.

Client Profiles

1. Company input form

Enter minimum available operational data. Added sections G, H and I make the logic more defensible. Required fields must be completed; optional blanks are still allowed.

A Facility, lease and budget constraints
B Data and IT readiness
C Inbound profile
D Storage, product and inventory profile
E Order picking profile
F Labour and current productivity
G Strategic objective

Important: The primary objective has the strongest influence, but the secondary objective is also considered in the final scoring.

H Growth profile
I Automation readiness preference

2. Assessment output

Run the assessment after editing inputs. The tool recommends a solution type based on the current form values, feasibility gates, selected primary objective and weighted scoring. Vendors are shown separately so the client can choose.

Hosted/static HTML. The Vendor Library is loaded from the same website; assessment inputs remain in the client browser.
